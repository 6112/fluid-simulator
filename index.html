Fluid Simulator

This Java program is a Swing application for simulating fluid physics. It was ported from a desktop application to an applet and, as a result, font rendering may be weird. It uses a particle system to mimic the appearance of a liquid inside a bottle.

You can use your mouse to pick up the fluid, and change the direction of the gravity, and other settings, from the menu on the right.

The color of the fluid indicates the pressure at a given point.

If the demo does not work in your web browser, you may want to try the standalone .jar file
